# Task 15 - Customer Segmentation (RFM Analysis)

This project performs RFM (Recency, Frequency, Monetary) analysis on the Online Retail II dataset.

## Files
- task15_rfm.ipynb : Jupyter notebook with full analysis
- rfm_segments.csv : Final customer segmentation output
- segment_actions.txt : Business actions per segment

## Steps Performed
1. Data cleaning and preprocessing
2. Date conversion
3. RFM calculation
4. RFM scoring using quantiles
5. Customer segmentation (Champions, Loyal, At Risk, Others)
6. Visualization of segments
7. Export results to CSV

## Dataset
Online Retail II dataset
